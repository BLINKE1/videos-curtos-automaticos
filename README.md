# Videos Curtos Automaticos

Pipeline automatizado para geração e upload de videos curtos no YouTube usando Claude AI.

## Stack
- **Script/Roteiro**: Claude API
- **Voz (TTS)**: Edge TTS (gratuito) / ElevenLabs (opcional)
- **Video B-roll**: Pexels API
- **Edição**: MoviePy + FFmpeg
- **Upload**: YouTube Data API v3
